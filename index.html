O que é GITHUB?

O GitHub é uma plataforma de hospedagem e compartilhamento de códigos-fonte e aplicações. Com sistemas completos para monitorar o desenvolvimento, ela também se tornou a mais usada no mundo da programação, atraindo desde empresas até especialistas independentes.

Como funciona o GITHUB?

O GitHub é gratuito, exigindo apenas um cadastro para utilizar todos os seus recursos. Os projetos são hospedados em “repositórios”, que podem ter acesso público ou privado. Desde a compra pela Microsoft, todos os usuários têm acesso ilimitado a repositórios particulares. Todo o gerenciamento dos projetos, como criação de novos repositórios, é feito via comandos do sistema Git. Você também pode gerenciá-los pelo site, mas normalmente os processos são executados via linha de comando do seu computador ou servidor

Como criar um repositótio no GITHUB?

Abra o terminal de comando do seu sistema operacional; Dentro da pasta do projeto, digite: git init O comando acima irá criar toda a estrutura básica do repositório; Para adicionar todos os arquivos alterados à fila de atualizações do repositório, execute o comando: git add . Antes de sincronizar as alterações, configure seu usuário do GitHub com os comandos: git config --global user.name "seu nome" git config --global "email no GitHub" Confirme as alterações com o comando: git commit -m "mensagem" no qual “mensagem” geralmente é um resumo das alterações. Adicione o remote, ou seja, o link para o servidor do seu projeto no GitHub: git remote add origin usuário no GitHub>/.git Por fim, envie as alterações com o comando: git push remote origin Se tudo deu certo, será exibido uma mensagem confirmando o envio. Você também pode checar na página do repositório do projeto no GitHub.com.
