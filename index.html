O que é GITHUB?

O GitHub é uma plataforma de hospedagem e compartilhamento de códigos-fonte e aplicações. Com sistemas completos para monitorar o desenvolvimento, ela também se tornou a mais usada no mundo da programação, atraindo desde empresas até especialistas independentes.

Como funciona o GITHUB?

O GitHub é gratuito, exigindo apenas um cadastro para utilizar todos os seus recursos. Os projetos são hospedados em “repositórios”, que podem ter acesso público ou privado. Desde a compra pela Microsoft, todos os usuários têm acesso ilimitado a repositórios particulares. Todo o gerenciamento dos projetos, como criação de novos repositórios, é feito via comandos do sistema Git. Você também pode gerenciá-los pelo site, mas normalmente os processos são executados via linha de comando do seu computador ou servidor
